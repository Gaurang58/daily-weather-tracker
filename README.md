# Daily Weather Tracker

A beginner-friendly Python project that collects daily weather data, stores historical records, and automatically pushes updates to GitHub.

## Planned features

- Fetch current weather data
- Store daily records in CSV
- Prevent duplicate entries
- Generate weather charts
- Automate daily collection
- Push updates to GitHub

## Technologies

- Python
- Open-Meteo API
- CSV
- Git
- Cron
